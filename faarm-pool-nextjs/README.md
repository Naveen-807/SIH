## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd faarm-pool-nextjs

# Install dependencies
npm install

# Run development server

# Build for production
```
